# Backlog — Mensagens que Vendem (MQV)

> Extensão do Chrome que turbina o WhatsApp Web para vendas.
> Desenvolvedor atual: Rodrigo (programador). Integração futura: Claude entra como apoio de programação.
> Última atualização: 2026-06-12

---

## Visão do Produto

Extensão do Chrome que altera o WhatsApp Web para funcionar como ferramenta de vendas:

- Banco de mensagens (templates prontos para reutilizar)
- Envio automático dessas mensagens
- Registro de informações do contato (não do usuário): status da negociação, fechou ou não fechou
- Geração de métricas, informações e gráficos
- Funil de atendimento

## Como usar este backlog

Cada ideia entra como uma linha na tabela da seção correspondente, com estes campos:

- **ID**: sequencial (MQV-001, MQV-002...)
- **Data**: quando a ideia foi registrada
- **Prioridade**: alta / média / baixa (definida quando houver)
- **Status**: ideia → discutir com Rodrigo → aprovado → em desenvolvimento → entregue
